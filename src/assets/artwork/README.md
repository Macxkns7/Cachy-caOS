# Artwork de Nest

Este directorio contiene las ilustraciones originales y recursos gráficos en alta resolución del proyecto.

## Contenido esperado

- Concept art
- Logos fuente
- Banners
- Splash screens
- Ilustraciones maestras

El sistema no debe depender directamente de estos archivos. Cuando un recurso se utilice en la interfaz, debe existir una versión optimizada dentro de `src/assets/icons/` u otra carpeta específica para tiempo de ejecución.
