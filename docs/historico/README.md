# Archivo histórico

**Estado:** Vigente como índice histórico  
**Última revisión:** 2026-07-16

Esta carpeta conserva documentación que explica cómo evolucionó Cachy-caOS, pero que ya no describe el sistema actual.

## Regla principal

Nada dentro de `docs/historico/` debe utilizarse como guía operativa sin contrastarlo con la documentación vigente.

## Era Omarchy

`era-omarchy/` contiene la descripción, configuraciones y estrategia de reconstrucción de la etapa en que Omarchy era la base del escritorio.

Esa etapa aportó:

- aprendizaje sobre Hyprland;
- traducciones y personalizaciones;
- PWAs y flujos de escritorio;
- cultura de snapshots;
- primeras reglas de documentación;
- motivación para construir una arquitectura propia.

Fue reemplazada por la dirección CachyOS limpio + Hyprland limpio + Nest, pero sigue siendo parte importante de la historia del proyecto.

## Política de archivo

Un documento se archiva cuando:

- su arquitectura fue reemplazada;
- ya no es seguro seguir sus instrucciones;
- sigue aportando contexto o decisiones útiles;
- existe una fuente vigente que lo reemplaza.

El archivo histórico no es una papelera: debe contener únicamente conocimiento que explique la evolución del proyecto.
