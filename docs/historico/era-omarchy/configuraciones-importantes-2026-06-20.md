# Documento histórico — Configuraciones importantes (era Omarchy)

**Estado:** Histórico
**Fecha original:** 2026-06-20
**Archivado:** 2026-07-16
**Reemplazado por:** `docs/configuraciones-importantes.md`

---

Este documento describía el sistema cuando Omarchy era la base del entorno visual.

Componentes registrados:

- Hyprland, Waybar y Walker administrados desde Omarchy.
- Waybar y Walker traducidos al español.
- Layout LATAM y keyd.
- EasyEffects.
- PWAs.
- Tema Aether, iconos Papirus y elementos EVA/NERV.
- Snapshots Btrfs.
- SDDM adaptado visualmente.

Se conserva para explicar el origen de varias decisiones y configuraciones que luego fueron reevaluadas durante la transición hacia una base CachyOS + Hyprland limpia y Nest.
